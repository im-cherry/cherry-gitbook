# 시간 복잡도

### 1. 빅오 표기법

알고리즘의 시간 복잡도를 나타내는 표기법

- O(n)
  {% tabs %}
  {% tab title="Javascript" %}

  ```javascript
  for(let i=0; i<n; i++){
      ...
  }
  ```

  {% endtab %}

- O(logn)
  {% tabs %}
  {% tab title="Javascript" %}

  ```javascript
  for(let i=0; i<n; i*=2){
      ...
  }
  ```

  {% endtab %}

- O(nlogn)
  {% tabs %}
  {% tab title="Javascript" %}

  ```javascript
  for(let i=0; i<n; i++){
      for(let j=0; j<n; j*=2){
          ...
      }
  }
  ```

  {% endtab %}

- O(n2)
  {% tabs %}
  {% tab title="Javascript" %}
  ```javascript
  for(let i=0; i<n; i++){
      for(let j=0; j<n; j++){
          ...
      }
  }
  ```
  {% endtab %}

### 2. 빅오 표기법에서 중요한 법칙

- 상수항은 무시한다.
- 가장 큰 항외에는 전부 무시한다.
