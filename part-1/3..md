# 시간 복잡도

### 1. 빅오 표기법

알고리즘의 시간 복잡도를 나타내는 표기법

* O(n)

```javascript
for(let i=0; i<n; i++){
    ...
}
```

* O(logn)

```javascript
for(let i=0; i<n; i*=2){
    ...
}
```

* O(nlogn)

```javascript
for(let i=0; i<n; i++){
    for(let j=0; j<n; j*=2){
        ...
    }
}
```

* O(n^2)

```javascript
for(let i=0; i<n; i++){
    for(let j=0; j<n; j++){
        ...
    }
}
```
